# Pipelined-RISC-V-Processor-RV32I-
This repository contains the RTL implementation of a 32-bit Pipelined RISC-V Processor based on the RV32I base integer instruction set. The design encompasses a datapath partitioned into fetch, decode, execute, and write-back stages, utilizing dedicated pipeline registers[cite: 5, 6, 7, 8] to manage instruction flow and control signals.
